# ER-Dashboard-Power-BI
Analysing 9,192 patient visits across 2 years of ER data, I built an end-to-end Power BI solution using a star schema data model and DAX measures to track KPIs including patient growth rate, wait times, and satisfaction scores. Patterns in peak traffic and departmental wait times pointed directly to where staffing needed to shift and how operations could be restructured for better patient outcomes.
